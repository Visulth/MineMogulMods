# DeletePickaxes

Deletes all pickaxes when deleting physics objects from the start menu (including ones in your inventory)

(Someone bought 1000 pickaxes)

## Version
- Mod: 1.0.0
- Tested with:
	- BepInEx: 5.4.23.4
	- MineMogul 0.1.0.3 Early Access

# FreezeGhost

Press H to freeze a buildable ghost/hologram

While frozen, can nudge horizontally with arrow keys, or vertically with page up or down.

## Version
- Mod: 1.0.0
- Tested with:
	- BepInEx: 5.4.23.4
	- MineMogul 0.1.0.3 Early Access

# Hide Quest HUD

Removes HUD text for active quests (they are still active and can be viewed in the Quests screen)

## Version
- Mod: 1.0.0
- Tested with:
	- BepInEx: 5.4.23.4
	- MineMogul 0.1.0.3 Early Access

# NoSupports

Prevents all buildables from building supports. 

Non-destructive; if the mod is removed all supports are created normally.

## Version
- Mod: 1.0.0
- Tested with:
	- BepInEx: 5.4.23.4
	- MineMogul 0.1.0.3 Early Access
